#### 4.12.5 EOL/PLP Failure Mode (Feature Identifier C2h) Set Feature

> **Section ID**: 4.12.5 | **Page**: 58-58

This Set Feature defines the mode to which the device shall transition at End of Life (EOL) or on failure
of the Power Loss Protection (PLP) circuitry.
