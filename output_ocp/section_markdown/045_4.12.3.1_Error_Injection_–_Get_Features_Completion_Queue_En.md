> **Section ID**: 4.12.3.1 | **Page**: 56-56

This Get Feature returns the set of error injections that are enabled on the device.  The attributes
specified in Section 4.12.2.2 - Device Panic Error Injection Type  are returned in Dword 0 of the
completion queue entry and the Error Inject data structure specified in Section 4.12.2.1 - Error Injection
Entry Data Structure is returned for each error injection in the data buffer for that command. If there are
no currently enabled error injections, the data buffer returned shall contain all zeros.  The device shall
clear to zero all unused entries in the Error Injection data structure.
