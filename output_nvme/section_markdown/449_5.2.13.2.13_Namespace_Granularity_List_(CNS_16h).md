###### 5.2.13.2.13 Namespace Granularity List (CNS 16h)

> **Section ID**: 5.2.13.2.13 | **Page**: 389-389

If the controller supports reporting of Namespace Granularity refer to the applicable NVM Express I/O
Command Set specification for details.
The controller shall abort the command with a status code of Invalid I/O Command Set if the Command Set
Identifier is not associated with an I/O Command Set that supports the Namespace Granularity List.
