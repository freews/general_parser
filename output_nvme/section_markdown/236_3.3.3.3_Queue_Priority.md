##### 3.3.3.3 Queue Priority

> **Section ID**: 3.3.3.3 | **Page**: 124-124

If the weighted round robin with urgent priority class arbitration mechanism is supported, then a host may
assign a queue priority service class of Urgent, High, Medium, or Low. If the weighted round robin with
urgent priority class arbitration mechanism is not supported, then the priority setting is not used and is
ignored by the controller.
