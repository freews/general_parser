###### 5.2.12.1.32.1.4 Invalid Placement Identifier (Event Type 3h)

> **Section ID**: 5.2.12.1.32.1.4 | **Page**: 312-312

This FDP event indicates that a write command specified an invalid Placement Identifier (refer to Figure
289 and Figure 290) due to an invalid Placement Handle or an invalid Reclaim Group Identifier.
