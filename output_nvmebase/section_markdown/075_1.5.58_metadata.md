#### 1.5.58 metadata

> **Section ID**: 1.5.58 | **Page**: 34-34

Metadata is contextual information related to formatted user data (e.g., a particular LBA of data as defined
in the NVM Express NVM Command Set Specification). The host may include metadata to be stored by the
NVM subsystem if storage space is provided by the controller. Refer to the applicable NVM Express I/O
Command Set specification for details.
