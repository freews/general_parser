###### 8.3.4.1.3 Clearing all Exported NVM Subsystems

> **Section ID**: 8.3.4.1.3 | **Page**: 737-738

Issuing a Clear Exported NVM Resource Configuration command (refer to section 5.4.1) clears all Exported
NVM Subsystems (refer to Figure 782).
Upon successful completion of a Clear Exported NVM Resource Configuration command there are no
Exported NVM Subsystems.
