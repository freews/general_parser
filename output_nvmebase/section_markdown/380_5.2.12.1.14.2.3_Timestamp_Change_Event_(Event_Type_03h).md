###### 5.2.12.1.14.2.3 Timestamp Change Event (Event Type 03h)

> **Section ID**: 5.2.12.1.14.2.3 | **Page**: 269-269

The Timestamp Change Event (refer to Figure 236) contains the current timestamp, reported in the event
header, and the timestamp from the time at which the timestamp was changed (i.e., the old timestamp).
