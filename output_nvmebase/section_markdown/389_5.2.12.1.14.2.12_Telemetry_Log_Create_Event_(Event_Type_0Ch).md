###### 5.2.12.1.14.2.12 Telemetry Log Create Event (Event Type 0Ch)

> **Section ID**: 5.2.12.1.14.2.12 | **Page**: 280-280

A Telemetry Log Create event may be created if the controller determines that a Telemetry Host-Initiated
log page (refer to section 5.2.12.1.8) or that a Telemetry Controller-Initiated log page (refer to section
5.2.12.1.9) has been generated.
