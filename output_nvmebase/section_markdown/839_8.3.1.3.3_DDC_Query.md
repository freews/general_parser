###### 8.3.1.3.3 DDC Query

> **Section ID**: 8.3.1.3.3 | **Page**: 701-701

A DDC may determine if a CDC is present by transmitting a query that includes a DNS PTR record (refer
to RFC 6763) with the name in the form of:
“ cdc. sub. nvme-disc.<protocol>.<domain>”.
The protocol field shall be set as shown in Figure 732.
The domain field shall be set as shown in Figure 734.
