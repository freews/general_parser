###### 5.2.12.1.32.2.1 Media Reallocated (Event Type 80h)

> **Section ID**: 5.2.12.1.32.2.1 | **Page**: 312-312

This FDP event indicates that user data contained in a Reclaim Unit was written by the host specifying a
Reclaim Unit Handle with a Reclaim Unit Handle type of Initially Isolated and was moved by the controller
to a different Reclaim Unit (e.g., controller performed garbage collection).
The Event Type Specific field is I/O Command Set specific. Refer to the applicable NVM Express I/O
Command Set specification to determine if this log page is supported and the definition of the Event Type
Specific field for this FDP event.
