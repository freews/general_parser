###### 5.2.26.1.9 Host Controlled Thermal Management (Feature Identifier 10h)

> **Section ID**: 5.2.26.1.9 | **Page**: 434-434

This Feature configures the controller settings for the host controlled thermal management feature, refer to
section 8.1.18.5. The host controlled thermal management feature uses Command Dword 11 with the
attributes shown in Figure 417.
If a Get Features command is submitted for this Feature, then the attributes shown in Figure 417 are
returned in Dword 0 of the completion queue entry for that command.
